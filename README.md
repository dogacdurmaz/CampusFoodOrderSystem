# Campus Food Order System

This project is created for Software Design Patterns homework.

The system is a simple campus food ordering application written in Java.

## Design Patterns Used

- Factory Pattern
- Singleton Pattern

## Features

- Email notification
- SMS notification
- Push notification
- Shared application configuration

## Java Files

- Main.java
- Notification.java
- EmailNotification.java
- SmsNotification.java
- PushNotification.java
- NotificationFactory.java
- AppConfig.java
- CampusFoodOrderService.java

## How to Run

1. Compile the Java files
2. Run Main.java
